Assistive Device
Design and innovation are meant to help people. At the very least, my designs should make someone’s life easier. The most obvious way to accomplish this goal is to address challenges people have on a daily basis. Assistive devices are intended to supplement a user's ability to do simple tasks that prove challenging to them for whatever reason. These devices can address disabilities, injury, or even just plain laziness.
Web Design
Part of this project is becoming familiar with webpage design. To accomplish this with little-to-no coding experience, I used a Bootstrap webpage template and accessed the code. By simply editing the code, copying and pasting elements, and altering settings, I can change the appearance and functionality of the template. In this way, I’ve created my own unique personal website.

First Print

Name Tag
To familiarize myself with the 3D printing process, I first made a simple name tag. The file for this print was provided, and similar tags can be found on open-source 3D printing sites. This was my first experience editing a printing file by simply changing the text from “Your Name” to “Josh Bub.”

Tourture Test

3D Benchy File
This is the 3D Benchy file open on the Prusa Slicer software. This program allows us to change print settings, specify filament type, and predict print time and failure points.
Test Result
The 3D Benchy turned out wonderfully. Aside from minor structural gaps in small details like the chimney, and some stringy plastic in the cabin, the Benchy was perfect. This highlights the capabilities of the Prusa Mini with which we will print design prototypes.

Assistive Device

Open-Source Files and Printing Process
One benefit of 3D printing is the incredible amount of free, open-source, and easy to print designs that can be found all over the internet. One such open-source site is called Makers Making Change, which contains designs for assistive devices that can be printed, laser cut, assembled with hardware, or some combination of those manufacturing modalities. Here, I found a simple bottle opener for those who lack the ability to grip things strongly.
Finished Product
This device is a simple and quick print job. This kind of design is ideal because the print needs some structural integrity to bend when squeezed. It is also convenient for me, as the print only took about half an hour. This particular design came with multiple size choices. I chose “40,” a size meant for typical plastic bottles.
